# LACEYjs
laceyjs is a small javascript library for module oriented development.

### Install with npm
  npm install lacey --save

### Install with Bower
  bower install lacey --save

### Install with Ruby
  gem install lacey-rails

or add to the gemfile:
```ruby
gem 'lacey-rails', '~> 1.0.0'
```
